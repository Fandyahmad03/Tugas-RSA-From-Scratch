# RSA From Scratch (Python)

Tugas Kriptografi - Implementasi RSA tanpa menggunakan library enkripsi.

## 📌 Deskripsi
Program ini mengimplementasikan algoritma RSA mulai dari:
- Key Generation
- Enkripsi
- Dekripsi
- Modular Exponentiation
- Extended Euclidean Algorithm

## 🔐 Algoritma
RSA (Rivest-Shamir-Adleman) adalah algoritma kriptografi asimetris yang menggunakan pasangan kunci publik dan privat.

## ▶ Cara Menjalankan
1. Pastikan Python sudah terinstall
2. Jalankan perintah:
   python rsa.py

## 💡 Contoh Output
Plaintext = 9  
Ciphertext = 70  
Decrypted = 9
